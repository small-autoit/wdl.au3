# wdl.au3
[WinDrawLib](https://github.com/mity/windrawlib) wrapper for AutoIt3.

### Basic types

```c++
Point
    x, y : float
```

```c++
Rect
    x0, y0, x1, y1 : float
```

```c++
Matrix
    m11, m12: float
    m21, m22: float
    dx, dy : float
```

```c++
PathSink
    data : ptr
    x, y : float

```
